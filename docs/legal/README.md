# KindFi Legal Documents (Starter Pack)

This package is a production-ready **starter** legal suite for KindFi.

The website footer pages (`/terms`, `/privacy`, `/cookies`, `/licenses`) are served from
`apps/web/content/legal/` — keep those copies in sync when updating the four published docs.

Included documents:

- terms-of-use.md
- privacy-policy.md
- cookie-policy.md
- donation-policy.md
- aml-kyc-policy.md
- risk-disclosure.md
- transparency-policy.md
- licenses.md
